# Low Level System Design

A comprehensive repository for learning and practicing Low Level Design (LLD) concepts, design patterns, and real-world system implementations.

## 📚 Repository Structure

### Core Concepts

#### Low Level Design (Basic Building Blocks)
Fundamental concepts and building blocks for creating robust low-level designs.

#### SOLID Principles
- Single Responsibility Principle
- Open/Closed Principle
- Liskov Substitution Principle
- Interface Segregation Principle
- Dependency Inversion Principle

#### Design Patterns

**Creational Patterns**
- Factory Pattern
- Abstract Factory Pattern
- Singleton Pattern
- Builder Pattern
- Prototype Pattern

**Structural Patterns**
- Adapter Pattern
- Bridge Pattern
- Composite Pattern
- Decorator Pattern
- Facade Pattern

**Behavioural Patterns**
- Observer Pattern
- Strategy Pattern
- Command Pattern
- State Pattern
- Chain of Responsibility Pattern

### 🚀 LLD Projects

This repository includes implementations of various real-world systems:

- **Pizza Factory** - Factory pattern implementation
- **SQL Query Builder** - Builder pattern for constructing SQL queries
- **Logger** - Logging system with multiple handlers and levels
- **Car Price Calculator** - Pricing engine with various calculation strategies
- **Weather Station** - Observer pattern implementation
- **Ticket Management System** - Event booking and management
- **Payment Processor** - Payment gateway integration system
- **Elevator System** - Multi-elevator scheduling and control
- **Car Parking** - Parking lot management system
- **Social Media** - Social networking platform design
- **WhatsApp** - Messaging application design
- **Amazon** - E-commerce platform design
- **PayPal** - Payment processing system

## 🎯 Learning Goals

- Master SOLID principles and their practical applications
- Understand and implement common design patterns
- Build scalable and maintainable low-level designs
- Practice real-world system design problems
- Develop clean, testable, and extensible code

## 🛠️ Technologies

This repository demonstrates LLD concepts that can be implemented in any object-oriented programming language.

## 📖 How to Use

1. Start with **SOLID Principles** to build a strong foundation
2. Progress through **Design Patterns** (Creational → Structural → Behavioural)
3. Apply learned concepts to **LLD Projects**
4. Each topic includes theory, examples, and hands-on implementations
